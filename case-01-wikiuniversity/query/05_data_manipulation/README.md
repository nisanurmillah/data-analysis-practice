# Data Manipulation (INSERT, UPDATE, DELETE)

Folder ini berisi query yang mengubah data.

## Operasi yang Dilakukan
- INSERT data customer, order, dan order_detail
- UPDATE negara customer
- UPDATE nama produk berdasarkan kategori
- DELETE order dan order_detail berdasarkan kondisi tertentu

## Insight
Query DML digunakan untuk simulasi perubahan data
dan memahami dampaknya terhadap analisis.
